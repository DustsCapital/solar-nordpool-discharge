# Solar Nord Pool Discharge Automation

A simple Python-based automation for Raspberry Pi to fetch 15-minute Nord Pool electricity prices (LV zone) and schedule SolaX EMS1000 battery discharges during peak slots for revenue optimization.

Owner: dustscapital

License: MIT

Clone and configure via config.py for your farm. Test with python main.py --test-today and python solar_discharge.py --test.
